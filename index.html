Guidelines

Accessibility (WCAG 2.1 AA) 無障礙設計指南

WCAG 2.1 Principles (POUR)

Perceivable

Content must be presentable in ways users can perceive.

Operable

Interface must be operable by all users.

Understandable

Content must be understandable by users.

Robust

Content must be robust for assistive tech.

Design Checklist

  • Color contrast: 4.5:1 for text, 3:1 for UI
  • Touch targets: 44×44px minimum
  • Focus indicators: 3px ring visible
  • Color independence: Never color alone

Developer Checklist

  • Semantic HTML: Proper elements
  • ARIA labels: For icon buttons
  • Keyboard nav: Tab through all
  • Skip links: Skip to main content
